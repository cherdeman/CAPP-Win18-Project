File: README.txt
Contains: Instructions for how to run Jupyter Notebook script to recreate transformed 2000 census data file

Steps:
1) Run the command "jupyter notebook" from this directory on the command line
2) When the kernel opens in your browser of choice, select 2000 Census Data Transformations.ipynb
   to open the desired script
3) The raw files are hosted on the Lexar USB drive provided, ensure that the path in cell 2
   directs to the appropriate place on your machine
4) While in the first cell, select Cell -> Run All Below from the dropdown menu
5) The new file will write out to csv in the location specified in the last cell,
   currently one directory level above the location of this script
